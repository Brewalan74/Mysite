Web Developer

Self-taught learning by using Internet resources. I started with 2 websites : The Odin Projectin English and w3schools, also in English. Its a good start for someone who want to learn by himself about the basics of HTML and CSS. The Odin Project gathers together lot of Internet knowledge about coding and it leads to different exercises to practice. w3schools provided many examples with very clear explainations and some exercises as well. For Javascript, in parallel with these 2 website, I also took a MOOC in French.

Javascript was less easier to learn in English so in addition to the 2 previous websites, I took an openclassrooms.com MOOC : "Learn to program with Javascript" in French.

I also took some course with Coursera in English, but not really updated and it was more difficult for me to understand everything about Javascript : "HTML, CSS and Javascript for Web Developers" and "Introduction to Web Development"
